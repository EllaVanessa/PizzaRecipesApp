# Pizza Recipes App

Android application that displays different pizza recipes with details and sharing feature.

##  Features
- Splash screen
- List of pizzas
- Pizza details (ingredients, description, steps)
- Share button to send recipe

## Technologies
- Java
- Android Studio
- XML (UI)
